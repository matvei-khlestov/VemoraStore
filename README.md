## Overview

VemoraStore is an iOS demo project that showcases the architecture and UI of a modern e-commerce application.

The project focuses on clean navigation, scalable architecture, and reusable UI components, with preparation for backend integration and real-world features such as user accounts, product catalog, maps, and local persistence.

## Key Features

- Modular screen structure
- Scalable navigation using Coordinator pattern
- State-driven UI with predictable data flow
- Reusable and extensible UI components
- Prepared architecture for backend integration
- Local data persistence and secure storage
- Map-based features and location handling

## Tech Stack

- Swift
- UIKit
- MVVM
- Coordinator
- Combine
- async / await
- Dependency Injection (Factory)
- Firebase SDK
- Core Data
- UserDefaults
- Keychain
- MapKit
- Core Location
- FileManager
- Unit Tests
- UI Tests

## Notes

This project is designed as a showcase of architectural decisions, navigation patterns, and UI composition that can be extended into a full production-ready application.
